PyCat

Pycat is a simple python hash bruteforcer.

It supports...

  • MD4/5
  • SHA1/224/256/384/512
  • SHAKE128/256
  • SHA31/224/256/384/512
  • NTLM

It was designed to run on lower hardware

Where hashcat may not work

PyManage

High quality, secure python password manager.

Features...

  • SHA256 hashing to ensure master password security
  • AES CBC 256 to encrypt passwords in the database.
  • SQLITE3 database for efficient storage of passwords
  • Easy to use setup
  • Simple WebGUI interface available

Designed to be my personal manager

To keep my more private accounts secure

Warlock

Python social media osint tool.

Features...

  • Multithreading to incrase scan speed
  • Allows for custom lists to be used
  • Checks for status code, title or page content
  • Spoofs user agents to form real looking requests
  • Grabs extra information on certain websites

Inspired by the sherlock osint tool

But improved in some ways

Reaper

Python based network testing toolkit

Features

  • Layer 4/7 DOS testing
  • Multithreading to run tests faster
  • General networking tools such as reverse DNS
  • UDP, TCP, SYN Layer 4
  • Socket debugger
